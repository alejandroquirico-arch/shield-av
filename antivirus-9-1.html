¿CÓMO FUNCIONA UN ANTIVIRUS?
Los antivirus modernos combinan tres técnicas principales para detectar amenazas:
  • Firmas de virus — comparación con base de datos conocida
  • Análisis heurístico — detecta comportamientos sospechosos
  • Sandboxing — ejecuta el archivo en entorno aislado
  • Machine Learning — modelos entrenados en millones de muestras
  • Análisis en la nube — hash lookup en bases de datos globales
🧬
ANÁLISIS HEURÍSTICO
Esta app aplica heurística estática que analiza características del archivo sin ejecutarlo:
  • Extensiones ejecutables de alto riesgo (.exe, .bat, .ps1)
  • Archivos con nombre genérico sospechoso
  • Tamaño inusual (muy pequeño o muy grande)
  • Doble extensión (factura.pdf.exe)
  • Entropía alta (posible cifrado/ofuscación)
🔑
HASHING CRIPTOGRÁFICO
El hash SHA-256 es una "huella digital" única de cada archivo. Si un solo byte cambia, el hash cambia completamente. VirusTotal mantiene una base de datos de hashes de millones de archivos maliciosos conocidos.
  • SHA-256: 64 caracteres hexadecimales
  • MD5: 32 caracteres (obsoleto para seguridad)
  • Inmutable: no se puede revertir
  • Determinístico: mismo archivo = mismo hash siempre
